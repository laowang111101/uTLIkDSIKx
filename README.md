# 项目概述 java1223

本项目是一款集成了安卓客户端与SSM（Spring、SpringMVC、MyBatis）后台架构的大学课程管理平台。提供完整的技术文档，但不包含部署服务。

# 技术栈

## 后台技术

- 架构：SSM（Spring、SpringMVC、MyBatis）
- 开发工具：IDEA
- 数据库：MySQL
- 数据库可视化工具：Navicat
- 服务器：Springboot自带的Apache Tomcat
- 主要技术：Java、SSM、MyBatis、MySQL、JQuery

## 前端技术

- 安卓开发工具：HBuilderX

## 开发环境

- 开发系统：Windows 10
- JDK版本：Java JDK 1.8
- 数据库版本：MySQL 8.0
